# Templates del equipo

Este directorio contiene templates reutilizables
para estandarizar cómo documentamos y colaboramos.

No todos los templates se usan siempre.
Cada uno indica explícitamente **cuándo debe usarse**.

## Templates disponibles

- **ADR**  
  Para decisiones técnicas relevantes y con trade-offs.

- **Experiment report**  
  Para registrar experimentos que informan decisiones.

- **Model card**  
  Para modelos que impactan decisiones reales.

- **Pull request**  
  Para estructurar todos los PRs del repositorio.

Los templates no son burocracia.
Son atajos compartidos para trabajar mejor.
